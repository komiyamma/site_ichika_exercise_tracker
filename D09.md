## 🌸 **9.3 HTMLバリデーションの活用**

HTMLフォームの入力フィールドには、**`required`属性**や**`type`属性**を使用して、**JavaScriptなしで入力チェック**を行います。これにより、**必須項目の未入力や不正な入力**を防ぐことができます。

### **初心者の悩み**：

「**HTMLバリデーション**ってどう使うのか分からない。JavaScriptで入力チェックをするのは面倒だし、簡単にできる方法がないかな？」

### **制作者の思考の軌跡**：

「**`required`属性**で必須項目を自動的にチェックでき、ユーザーが必須項目を未入力のまま送信することを防げる。これで、入力エラーを防げるので、コードが簡潔になる。」

#### **HTMLバリデーションの追加**

フォーム項目には、以下のように**`required`属性**を追加して、必須項目を**HTMLで簡単にバリデーション**できるようにします。

```html
<!-- 必須項目にrequiredを追加 -->
<label for="type" class="form-label">種類</label>
<input id="type" type="text" class="form-control" required>

<label for="date" class="form-label">日付</label>
<input id="date" type="date" class="form-control" required>
```

これにより、ユーザーが**未入力のまま送信**しようとすると、ブラウザが**エラーメッセージ**を表示して、送信を停止します。

function renderEntryTable() {
