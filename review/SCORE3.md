### 3. 配列メソッドの実践的な学習設計 ★★★★★

**Day 9での工夫（最新改善点）**:

#### ✅ forループを基本として提示
```javascript
let tableHtml = '';
for (const entry of entries) {
    tableHtml += `<tr>...</tr>`;
}
```

#### ✅ map/joinを「別の書き方」として紹介
```javascript
const tableHtml = entries
    .map(entry => `<tr>...</tr>`)
    .join('');
```

#### ✅ パフォーマンス神話の解消
> 「`map().join()`にしたから速くなる」わけではありません。
> むしろこのシーンでは`map + join`の方が「ほぼ確実に遅く」なります。

**この設計の教育的意義**:
1. **forループで基本を確実に習得**させる
2. **mapは「読みやすさ重視の別解」**として提示
3. **パフォーマンスの誤解を解消**（重要！）
4. **「自分にとって読みやすい方を選ぶ」**という実践的判断力を養う

**他の配列メソッド使用例**:
- `filter`: Day 11で日付絞り込みに使用
- `sort`: Day 9で新しい順の並び替えに使用

**評価**: 💯 初学者が陥りがちな誤解まで丁寧にフォロー
